# 标准中文标点与文学节奏约束

本文件用于约束长篇文学项目中的中文正文、对白、审查报告、风格提示词、导出文本和用户可见说明。它不是某一种作家风格，而是所有文风之下的基础排版卫生层和叙事节奏底线。

## 基础规则

- 中文叙事、对白、报告和说明默认使用全角中文标点：，。；：？！、（）《》“”‘’。
- 中文正文中不要混用英文逗号、句号、冒号、分号、问号、感叹号或英文引号；英文缩写、URL、文件路径、代码、schema 字段和小数除外。
- 省略号统一使用“……”，不要使用“...”或连续句点。
- 破折号统一使用“——”，不要用“--”或单个“—”表达中文停顿。
- 疑问号、感叹号通常只使用一个；避免“！！！”“？？？”“?!”等网感化连续标点，除非用户明确要求实验性文本或角色文本确有格式依据。
- 横排文学正文的直接引语统一使用中文双引号“”；引语内再引语使用中文单引号‘’。不要在不同章节混用「」『』、﹁﹂﹃﹄、｢｣等竖排/角引号，除非项目记录了明确的版式例外。
- 不要在中文标点前留空格，不要连续堆叠逗号、句号、顿号、冒号或分号。

## 文学节奏规则

标点不是装饰，也不是“把话切开”的机械工具。标点应服务叙述距离、镜头调度、人物心理、动作连续性和因果推进。

- 句号用于完成一个语义落点、镜头落点或心理落点。不要把同一组连续动作切成一串过短句号；若动作仍在同一感知链、因果链或情绪波内，优先使用逗号、分号、顿号或重写句法承接。
- 逗号用于未完成的关系：动作延续、视线移动、心理补充、因果递进。不要用逗号串接过多彼此独立的动作；长逗号链应拆为句群，或用分号区分并列层级。
- 破折号只可作为需要复核的强停顿信号。普通正式正文原则上不用破折号；孤立出现时必须确认它承担话语中断、真实插入或必要解释，不是把“但是/而是/于是”换成装饰性停顿。超过约 2% 叙事单元密度、同段反复出现，或构成“不是……——是……”等生硬对照时，必须修订。
- 若文风证据中存在高破折号节奏，只能抽象其停顿、讽刺、信息反转或话语中断功能，不把破折号密集使用写成生成模板；平台 agent 必须逐句判断，不得按正则批量删改。
- 转折不应主要依赖“但是、然而、于是、然后、突然”等显性连接词。优先用人物动作、视线变化、物象回声、信息差和句法重心制造转折；显性转折词只在逻辑关系需要被点明时使用。
- 长短句应围绕注意力变化调度：观察和铺陈可以稍长，判断、发现、拒绝、危险逼近可以短，但短句必须有信息落点，不能只是模型化碎句。
- 对白提示语的标点服务说话动作和语气。不要让提示语反复用同一种“他说。”式句号收束；可通过动作、停顿、沉默和环境反应形成段落节奏。

## 不应机械替换的区域

- Markdown 表格、列表标记、链接和代码块。
- JSON、YAML、TOML、schema 字段、文件路径、命令、URL、模型名和英文术语。
- 伪记录或档案体中作为证据保留的原始引用，但正文叙述和整理说明仍应使用标准中文标点。

## 与文风的关系

文风学习可以调整标点的密度、停顿感和句法节奏，但不能突破本基础规范，也不能把模型习惯误判成作家风格。若某个 Style Skill 发现非标准标点、极短句群、密集破折号或高度显性的转折词，平台 agent 必须抽象其叙事功能，并把风险写入审查记录或“需要人工确认”。机械“不是……而是……”及其破折号/句号变体不得被判断为合理修辞。

## 自动清洗边界

脚本可以做安全排版规范化，例如统一引号样式、识别英文标点混入、提示省略号或破折号格式错误。脚本不得做语义级“去 AI 腔”改写：不得按正则删除“不是”、把“不是 A——是 B”替换成普通陈述、批量删除破折号、替换心理判断或改变叙事者纠偏结构。这些改动必须由平台 agent 逐句语义复核，因为错误改写会造成否定关系消失、信息纠偏失效或人物心理反转。

## 审查要求

场景候选、正文草稿、章节导出和 DOCX 成品进入 ready/release 前，应至少检查：

- 中文句子中是否混入英文标点。
- 是否存在“...”而非“……”。
- 是否存在“--”或单个“—”充当中文破折号。
- 是否存在连续感叹号、问号或中英混合疑问/感叹标点。
- 是否存在章节间引号风格不统一，尤其是“”与「」『』、﹁﹂﹃﹄或｢｣混用。
- 中文标点前是否有空格。
- 是否把 Markdown、JSON、路径、URL 或 schema 里的必要 ASCII 符号误当作正文标点问题。
- 是否存在句号密度过高、短句切分过碎的问题。
- 是否存在逗号链过长、句内关系被机械串接的问题。
- 是否存在破折号滥用，把“——”当成普通转折或强调装饰的问题。
- 若破折号密度高，是否超过约 2% 叙事单元密度，或是否构成“不是……——是……”等生硬对照变体。
- 是否存在“但是、然而、于是、然后、突然”等显性转折词过密，使转折显得生硬的问题。
